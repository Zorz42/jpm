jpm install [packages] - install packages and their dependencies
jpm remove [packages]  - remove packages and their dependencies
jpm list               - list all installed packages
jpm listall            - list all packages in main repository
jpm cleanup            - remove unused dependencies and invalid files
jpm upgrade            - upgrade jaclang to latest version
jpm updatedatabase     - update database for listall and autocompletion
jpm uninstall          - uninstall jac software
